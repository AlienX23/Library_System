## ‚ú® Overview

This system is a basic console-based application designed to manage the available books in a small library. It uses **Object-Oriented Programming (OOP)** principles to handle book data.

## üöÄ Key Features

The system provides the following core library management functionalities:

* **View All Books:** Check the complete list of all books currently stored in the system.
* **Book Checkout:** Allow users to check out an available book using its **Book Name** or **Book ID**.
* **Book Return:** Process the return of a book using its **Book Name** or **Book ID**.
* **Search:** Find specific books using criteria such as **Book ID**, **Book Name**, or **Author Name**.
* **Add New Book:** Easily add new book records to the library system.

## üóÉÔ∏è Data Structure

Each book is stored as a custom **Book object** within an **`ArrayList`**. The following properties are maintained for every book record:

* **Unique Book ID**
* **Book Name**
* **Author Name**
